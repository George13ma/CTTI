# CTTI — Crypto Trader Type Indicator

## 1. 产品概述

| 字段 | 内容 |
|------|------|
| 产品名称 | CTTI — Crypto Trader Type Indicator |
| 一句话描述 | 加密货币交易人格测试，8题得出16种交易者类型 |
| 线上地址 | https://tradertest.vercel.app/ |
| 技术栈 | 纯 HTML/CSS/JS 单文件，无框架 |
| 作者 X | @George13ma |

---

## 2. 目标用户

**核心用户**
- 有合约交易经验的加密货币散户
- 亏过钱但还在坚持，想了解自己交易行为模式的人

**次要用户**
- 刚入场的新手，想提前了解自己风格

**排除用户**
- 专业机构交易员（产品对他们无娱乐价值）

**用户核心痛点**
- 不清楚自己为什么亏钱
- 缺乏对自身交易行为的认知

---

## 3. 产品价值主张

**解决什么问题**
用娱乐化方式帮助交易者认识自己的交易人格，引发共鸣和反思。

**核心体验**
测完之后：「这说的就是我」→ 忍不住分享给同类

**变现路径（当前）**
流量积累 → 小红书/X 内容传播 → 后续付费社群或周边

## 7. 分享功能说明

### 当前状态
- 分享按钮 UI ✅
- 复制文字到剪贴板 ✅
- 调用 navigator.share（手机原生） ✅
- 生成卡片图片 ❌ 待完成

### 分享卡片格式
竖版卡片，内容从上到下：
1. 品牌标识「CTTI」+ 段位徽章
2. 角色图片（images/{TYPE}.png）
3. 人格代码大字 + 子评级（如 S+）
4. POTENTIAL 标签 + 中文描述
5. NAME 中文名 + TYPE 风格描述
6. 星级评分 + 网址

实现方式：用 `html2canvas` 或 `dom-to-image` 渲染卡片 → 生成图片 → 跳转 X 发布

## 8. 结果页模块顺序

1. 段位标签（tier.label）
2. 角色头像（正方形，无边框，背景透明）
3. 人格代码 + 中文名 + 一句描述
4. 四维度评分卡片（4行）
5. 段位卡片
6. 人格解读（r.interpretation）
7. 潜力评分星星
8. 优势 / 陷阱 / 建议三张卡片
9. 友情提示 + 作者链接
10. 统计数据
11. 分享按钮 + 重新测试按钮
